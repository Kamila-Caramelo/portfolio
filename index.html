Projeto: Landing Page

Esta landing page é um projeto desenvolvido em estudo, foi realizado com HTML e CSS. Ela possui interação com o usuário, mesmo não utilizando outro tipo de tecnologia. Menu interativo tanto para desktop, quanto para mobile e a seta na parte inferior onde sutilmente informa o usuário que abaixo poderá ter acesso a mais conteúdo. Responsivo pronto, desenvolvido com cuidado para ficar ótima a resolução em qualquer tipo de dispositivo.
